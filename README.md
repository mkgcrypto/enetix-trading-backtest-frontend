# enetix-trading-backtest-frontend

Dash UI for the BB + KC + RSI backtester.

- Run locally: `python frontend/ui/dash_app.py`
- Configure backend: set `BACKEND_URL` (default `http://localhost:8000`)
- Tests: `pip install -r requirements/test.txt` then `pytest`

See the main repo `enetix-trading-backtest` for overall docs and local docker-compose usage.
